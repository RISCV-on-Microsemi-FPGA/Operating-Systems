# RISCV-Creative-Board
This folder contains the SoftConsole project with the project folder structure as used by uCOS stack. This project uses "virtual folders" and "linked resources".

The 'RISCV-Creative-Board\Micrium' folder contains the Micrium uCOS code.

The 'RISCV-Creative-Board\Microsemi' folder contains the BSP for the Mi-V hardware platform and the SoftConsole project.
